\# HR Employee Attrition EDA



\## Overview



This project analyzes employee attrition using the IBM HR Analytics dataset. The goal is to identify factors contributing to employee turnover and provide actionable business insights.



\## Tools Used



\* Python

\* Pandas

\* NumPy

\* Matplotlib

\* Seaborn

\* Jupyter Notebook



\## Analysis



\* Data Cleaning

\* Exploratory Data Analysis (EDA)

\* Attrition Distribution

\* Department-wise Analysis

\* Overtime Analysis

\* Correlation Analysis



\## Key Findings



\* Employees working overtime show higher attrition.

\* Lower-income employees are more likely to leave.

\* Attrition varies significantly across departments.

\* Younger employees tend to have higher turnover rates.
